# Smart Progress Tracker

A full-stack application for tracking personal progress, tasks, and skill development.

## Features

- User authentication
- Task management
- Weekly planning and goal setting
- Progress tracking and analytics
- Dashboard with charts and insights
- Job readiness scoring

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express
- MongoDB
- Mongoose

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation

1. Clone the repository
2. Install dependencies for client and server:
   ```
   cd client
   npm install
   cd ../server
   npm install
   ```
3. Set up environment variables in server/.env
4. Run the application:
   ```
   # In server directory
   npm run dev
   # In client directory
   npm run dev
   ```

## Usage

- Register/Login to access the dashboard
- Log tasks and track progress
- Set weekly goals and monitor achievements
- View analytics and charts
